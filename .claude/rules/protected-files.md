# Protected Files

These files control project configuration and governance. Modifications require HITL confirmation.

| Category | Files |
|----------|-------|
| Project Config | `CLAUDE.md`, `README.md` |
| Claude Config | `.claude/**` |
| Documentation | `docs/**` |
| DB Migrations | `backend/migrations/**` |
| Docker Config | `docker-compose.yml`, `Dockerfile` |

Meta file changes require explicit user approval before modification.
